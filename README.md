# S501-Groupe-Silksong-Projet-Vapeur
Une application mobile permettant la recommandation d’éléments à l’utilisateur sans qu’il n’ait besoin de se connecter.<br>
On utlisie des jeux vidéos comme données, et les Tags pour faire des recommandations.

## installation

## 1. Installer les dépendances

   ```bash
   npm install
   ```

## 2. Lancer l'application

   ```bash
   npx expo start
   ```
Pour l'affichage le plus proche du produit final il faut scanner le code QR avec [expo go](https://expo.dev/go)
